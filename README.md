# AI Detection Systems

A collection of Artificial Intelligence, Machine Learning, and Computer Vision projects focused on detection, classification, and intelligent data analysis using Python.

## About This Repository

This repository contains projects that demonstrate different AI-based detection and classification techniques. The projects cover computer vision, text classification, spam detection, and fraud detection while exploring practical applications of machine learning.

More AI detection projects will be added as I continue learning and building intelligent systems.

## Featured Projects

### Fake Profile Detector

Detects suspicious social media profiles using machine learning techniques and profile information.

### Object Detection System

Identifies and classifies objects from images using computer vision techniques.

### Fake News Detector

Classifies news articles as genuine or fake using machine learning and text analysis.

### Spam Email Detector

Detects spam emails based on email content using machine learning classification techniques.

## Installation

Clone the repository:

```bash
git clone https://github.com/maharukhh/ai-detection-systems.git
cd ai-detection-systems
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

## Running a Project

Navigate to any project folder and run the project.

```bash
cd project-folder
python main.py
```

Refer to the individual project README files for project-specific instructions.
