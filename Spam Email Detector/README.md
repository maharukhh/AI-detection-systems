# Spam Email Detector

A simple Python application that detects spam emails using keyword-based filtering techniques.

## Objective

The objective of this project is to identify whether an email message is spam or not by checking for commonly used spam-related keywords.

## Features

* Spam email detection
* Keyword-based filtering
* Interactive user input
* Simple and easy-to-understand implementation

## Technologies Used

* Python

## How It Works

1. Create a list of commonly used spam words.
2. Accept an email message from the user.
3. Convert the message to lowercase.
4. Search for spam-related keywords.
5. Classify the email as either **SPAM** or **NOT SPAM**.
